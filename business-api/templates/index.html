Employees

To get all employees, send a GET request to {{url}}employees/all

To get an employee, send a GET request to {{url}}employees/<id>

To add an employee, send a POST request to {{url}}employees along with a JSON with the name, city and country

To update an employee, send a PUT request to {{url}}employees/<id> along with a JSON with the name, city and country

To delete an employee, send a DELETE request to {{url}}employees/<id>
